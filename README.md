## :rocket: Getting started
``` 
  # Clone the repository:
  $ https://github.com/IffatAhmedk/pokedex/

  # Enter directory:
  $ cd pokedex
  
  # Install dependencies:
  $ flutter pub get
  
  # Run App:
  $ flutter run
```
